# 🛡️ Laboratorio de Ciberseguridad – Explotación de Metasploitable

Bienvenido a un laboratorio de pruebas de penetración realizado en un entorno controlado utilizando Kali Linux y Metasploitable.

## 🎯 Objetivos
- 🔍 Identificar servicios vulnerables
- 🛠️ Explotar vulnerabilidades conocidas
- 🏆 Obtener acceso root en el sistema objetivo

## 🧰 Herramientas Utilizadas
- 🛰️ Nmap – Escaneo de puertos y servicios
- 🗂️ Searchsploit – Búsqueda de exploits públicos
- 💣 Metasploit Framework – Explotación automatizada

## 🚩 Resultados Clave
- Vulnerabilidad explotada: Samba 3.0.20
- Acceso root conseguido en el sistema objetivo
